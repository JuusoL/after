
## Prerequisites

- Node.js and npm (tested on 6.5.0 and 3.10.7 respectively)

## Develop

    npm install
    npm start

Server will run at `http://localhost:8080/`

To build and test a distributable:

    npm run build
    cd dist && python -m SimpleHTTPServer
